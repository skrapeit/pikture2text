pikture2text
============

## Prerequisites

* jdk >= 8 installed

## Recommended

* gdub installed

## build project

    gw build
    
## run tests

    gw check
    
## run project

    gw bootRun
    
## build docker image

    gw dockerBuildImage
    
### start application as docker container

    docker run -p 8080:8080 pikture2text
